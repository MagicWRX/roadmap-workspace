# Images Directory

This directory contains the images used in the business template.

## Required Images:

1. **computer-repair-hero.jpg** (1200x600px recommended)
   - Hero section image showing computer repair service
   - Should represent professional computer/laptop repair
   - High-quality, professional photo

2. **profile-picture.jpg** (300x300px recommended)
   - Professional headshot of the business owner
   - Square aspect ratio for circular profile display
   - Clear, professional appearance

## Image Guidelines:

- Use high-quality images (minimum 72 DPI for web)
- Optimize images for web (use tools like TinyPNG)
- Hero image should be landscape orientation
- Profile picture should be square
- File formats: JPG, PNG, or WebP
- Keep file sizes reasonable for web loading

## Placeholder Images:

Until you add your own images, you can use placeholder services like:
- https://picsum.photos/ for general images
- https://via.placeholder.com/ for specific dimensions
- https://unsplash.com/ for high-quality stock photos
